# Student-Grading-System-C
# C ile Öğrenci Başarı Değerlendirme Sistemi

Bu proje, kullanıcının girdiği vize, devamlılık ve proje notlarını toplayarak öğrencinin dönem sonu başarı durumunu hesaplayan temel bir C konsol uygulamasıdır. 

---

##  Nasıl Çalıştırılır?

Herhangi bir C derleyicisinde veya terminal üzerinden GCC kullanarak projeyi hızlıca çalıştırabilirsiniz:

```bash
# Derleme
gcc main.c -o ogrenci_sistemi

# Çalıştırma
./ogrenci_sistemi
